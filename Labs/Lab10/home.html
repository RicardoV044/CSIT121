Web Standards Presentation: Geolocation API

What are Web Standards

Web standards are rules and guidelines that ensure web sites and other web technology are consistent and accessible on all devices and browsers. These web standards are made and maintained by organizations like the World Wide Web Consortium (Elcom, n.d.).

Purpose of this webpage

The purpose of this webpage is to inform on the web standard, Geolocation API.
